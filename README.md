## 내가 생각하는 서비스의 틀
1. 맨 처음 메뉴들을 관리자 페이지에서 등록한다.
2. 테이블 개수를 맞게 설정한다. ( 빈 데이터의 테이블들이 생성된다 이걸 서버에 전송하나)
3. 그 테이블 번호의 자리에서 주문이 온다.
4. 주문내역에는 { 테이블 번호, 주문내역, 요청사항 } 등이 있다.
5. 그 주문내역을 번호에 맞게 테이블자리에 뿌려준다.
6. 그 주문내역을 순차적으로 주문내역 페이지에 쌓아준다.
